# solana 扫链服务

## 功能点

- 支持扫链，支持自定义高度和默认最新高度。
- 关注的钱包地址，从数据库中读取。
- 扫描到主链币交易和代币交易都记录到数据库中，并且把交易信息投递到kafka中。
- 提供rpc接口，用于查询交易信息。
- 提供websocket接口，用于实时推送交易信息。
